# 楔入灯塔

许岚 在 古代灯塔群外壳舱段 完成了 物理占领与供能接管

塔环-07的外壳舱段像一段被遗忘的骨头，金属表皮被裂潮带的微陨砂磨出灰白的沟槽。许岚踩着磁靴贴在弧壁上前行，呼吸面罩里传来自己压得很低的笑声——不是轻松，是一种“还活着就得把账做完”的反射。

“外壳舱段封闭完成。”她把锁销打入最后一个接口，手腕终端弹出红色条目：**物理占领：已确认；供能路径：未建立。**

顾闻骁的身影在她身后浮现，战术灯扫过舱段中央那座古老的供能栈。它不像人类的设备，有一种不合规的优雅：层层叠叠的黑色陶瓷板，边缘有细到几乎看不见的光线流动。

“你确定这是‘供能栈’？”顾闻骁问。

“我确定它会吃电。”许岚蹲下，拆开维护盖板，露出一排本地维护端口——形制古怪，但针脚排列有某种对称的诚实。“接口在这。人类能插进去，说明它至少愿意装作兼容。”

她把转接器压上去，终端震了一下，弹出提示：**LOCAL MAINTENANCE ACCESS — GRANTED（限制级）**。下一行却像冷水：**CERT KEY — NOT FOUND**。

顾闻骁看着那行字，没有骂人，只是把它记进眼神里。“没有认证密钥，就只有占领，没有合法。”

“你要合法，先得活着。”许岚把备用电源包并联，按下手动合闸。舱段里一瞬间寂静得像真空，然后供能栈深处传来低频嗡鸣，像一头醒来的兽压着喉咙喘气。指示灯从暗到亮，沿着塔环外壳的电力母线一路点燃，像有人在黑暗里点了一圈火。

曲率潮汐监测面板跳出数据流：裂潮带的窗口预测曲线从混乱的毛刺变成可读的波形，误差带缓慢收敛。紧接着，信标发射状态从“离线”翻到“低功率自检”，再到“外壳级广播”。

顾闻骁轻轻吐出一口气，像把一条看不见的绳子重新绑回手上。“第七舰队能在窗口里走了。”

“能走，不等于不会被税盟说成‘危险航行’。”许岚抬手把实时供能日志和占领证据链封存进加密盒，标签却写得像审计报告：**塔环-07外壳舱段供能接管记录（可回滚）**。“我会让他们看见我们在维护，而不是在盗用。哪怕他们只看得见我们想让他们看见的部分。”

她的终端忽然闪过一段异常回波——兼容层残片的影子，在信标内部结构里像一层薄薄的雾。许岚的指尖停住，眼神变得更冷。

“又来了？”顾闻骁捕捉到她的停顿。

“残片回波。”许岚把功率限制往下压了一格，像给一只陌生机器扣上嘴套。“越深层越像‘核心信号’的门口。我们现在只拿到外壳的控制权，别急着敲里面的门——敲错一次，就可能把整个舰队拖进共同梦境里。”

顾闻骁沉默片刻，目光落在她屏幕上一项尚未开启的菜单：**AUDIT CACHE / READ**。

“那里面有裁决痕迹。”他说，声音平稳得像宣读军令，“税盟的阈值提高过，或者记录被改过。我们需要一条能谈判的证据链。”

“我知道。”许岚把手悬在“读取”上方，却没有按下去，“但我得先弄清楚，读缓存会不会触发更深层污染。我要一套不会把人塞进模型里就宣布合理的读取流程。”

她正要调出沙箱隔离参数，舱段里忽然响起一声细微的“滴”——不是人类设备的提示音，更像某种古老协议的唤醒。信标状态栏的最底部，多了一行她没见过的字：

**CORE SEGMENT 01 — HANDSHAKE REQUESTED**

同时，外部通信频道跳入一条压缩得过分干净的短报文，来源标记被抹去，只剩一句话：

——**别读缓存。审计会先读你。**
