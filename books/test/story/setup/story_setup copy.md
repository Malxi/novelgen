# 《星冕远征》

## Story Setup

### Genre(s)
- 硬科幻
- 太空歌剧
- 军事科幻
- 政治阴谋
- 群像史诗


### Core Premise
公元26世纪，人类凭借“曲率潮汐引擎”终于跨出太阳系，却在首次深空殖民中发现：银河并非空白版图，而是一张被古老“航道税制”与失落文明遗产共同编织的网。为夺取生存空间与航道主权，人类建立“星冕远征军”向外推进；然而每一次胜利都以更深的代价交换——殖民地叛乱、军工财阀操控、外星联盟的围堵、以及来自银河核心的未知信号。主角团从一支被当作弃子的边境舰队起步，在谎言与炮火中一步步撬开银河秩序，最终必须回答一个问题：征服银河，究竟是人类的出路，还是自我毁灭的加速器？

### Theme
扩张与自省：当文明拥有跨星系的力量，真正的敌人往往不是外族，而是自身的恐惧、贪婪与选择。

### Story Rules/Constraints
- 超光速航行依赖“曲率潮汐引擎”，必须锚定“潮汐航道”；航道由自然引力潮汐与人工灯塔共同稳定，灯塔可被摧毁、劫持或污染。
- 信息传递受限：跨航道量子通信只在“稳定窗口”可用，窗口之外只能靠信使舰或延迟中继，造成战略误判与政治操弄空间。
- 资源决定扩张速度：反物质、稀有同位素与“潮汐晶核”是战略物资；补给线越长，叛乱与走私越猖獗。
- 外星势力并非‘万能科技’，其优势体现在社会结构与长期布局；人类的优势是工业规模与战术适应速度。
- 古代遗产技术（遗构）不可直接复制，只能通过“兼容层”逐步驯化；每一次驯化都伴随不可预测的副作用。
- 战争有政治边界：大规模毁灭性武器会触发银河范围的‘制裁协议’，迫使各方以代理战争、封锁与暗杀替代全面灭绝。


### Target Audience
喜欢宏大世界观、舰队战争、政治博弈与强节奏反转的成人读者与青年读者；偏好《银河帝国》《三体》《太空堡垒》式史诗感的人群。

### Tone/Style
宏大、冷峻、紧迫；战场热血与权谋阴影并置，胜利常带苦涩代价。

### Narrative Tense
过去时

### POV Style
多视角第三人称（舰队指挥官、情报官、工程师、殖民领袖、外星使节交替推进）

## Storylines

### 主线：星冕远征——从边境弃子到银河破局者
- **Type**: 主线
- **Importance**: 10/10
- **Description**: 一支被派往“最不可能成功”航道的边境舰队意外夺得关键灯塔群，成为人类向银河拓展的楔子。舰队在连番会战中不断升级、吞并与分裂，同时被母星议会与军工财阀当作棋子。最终他们必须在‘继续扩张’与‘重塑秩序’之间作出选择。

### 支线：殖民地火种——自由与秩序的内战
- **Type**: 政治/社会
- **Importance**: 8/10
- **Description**: 前线殖民地因补给、税制与战时管制爆发连锁叛乱。主角团既要镇压，又要理解殖民者的生存逻辑；叛军背后还潜伏外星势力与财阀代理人，推动人类从内部崩解。

### 支线：灰网情报战——看不见的战线更致命
- **Type**: 谍战/阴谋
- **Importance**: 8/10
- **Description**: 情报官带领“灰网局”在量子窗口与信使舰之间周旋：篡改战报、制造假航道、策反敌方灯塔师。每一次情报胜利都可能换来更大的道德债务与更危险的真相泄露。

### 支线：遗构回声——古老文明的墓志铭
- **Type**: 探索/悬疑
- **Importance**: 7/10
- **Description**: 工程师团队在废墟星环中找到遗构“兼容层”，它能让人类舰船临时突破航道限制，却会在船员梦境中投射‘核心信号’。随着遗构驯化加深，人类逐渐发现银河秩序曾被某种‘看门机制’重启过无数次。

### 人物弧光：指挥官的王冠——胜利的重量
- **Type**: 人物成长
- **Importance**: 9/10
- **Description**: 主角指挥官从“求生”走向“称王”，不断被迫牺牲他人来换取战略优势；当他发现自己也成为被塑造的符号时，必须决定是做历史的工具还是打碎工具箱的人。

### 人物弧光：外星使节的双面誓言——和平或陷阱
- **Type**: 人物成长/跨文明
- **Importance**: 7/10
- **Description**: 一名外星联盟使节表面推动停火，暗地却利用人类扩张去击穿另一古老强权的封锁。她在家国责任与对人类个体的理解之间撕裂，最终选择将关键真相交给谁。


## Premises

### 曲率潮汐航道与灯塔体系 (科技/世界规则)
人类超光速并非随意跃迁，而是沿“潮汐航道”滑行。航道由星系引力潮汐构成，必须靠人工灯塔稳定与标定。灯塔的控制权等同于税权、军权与生存权，是银河争霸的核心资产。

**Progression System:**

**Level 1: 近恒星短航道**
- Description: 只能在相邻恒星系间进行低风险跳跃，航道窗口短，误差可控。
- Requirements: 拥有基础曲率引擎与单座灯塔校准；航行员通过“潮汐测算”认证。

**Level 2: 中程节点航网**
- Description: 以多灯塔串联形成“节点网”，可跨越数十光年，但必须按节点停靠校正，速度受限。
- Requirements: 建立至少三座灯塔并实现量子窗口同步；舰队需要专职灯塔师团队。

**Level 3: 深空干线航道**
- Description: 稳定窗口变长，可进行舰队级别的连续滑行，战略投送能力骤增；同时更容易遭到灯塔污染与航道伏击。
- Requirements: 夺取或自建“干线主灯塔”；掌握航道反干扰算法与潮汐晶核供能。

**Level 4: 无灯塔漂移航行**
- Description: 在没有灯塔的区域短时间维持曲率泡，绕开封锁线与税制，成为游击与奇袭的关键能力，但事故率极高。
- Requirements: 引擎升级为“潮汐回响核心”；船员需接受神经稳定训练并签署高风险协议；必须有遗构兼容层支持。

**Level 5: 核心潮汐折叠**
- Description: 借助银河核心的巨引力潮汐实现一次性超远程折叠，足以改写战争格局；但会触发“看门机制”的监测与反制。
- Requirements: 完成遗构兼容层完全驯化；集齐三段“核心信号”密钥；必须在政治上获得至少一个外星联盟的公开担保以避免制裁协议联合围剿。


### 星冕舰队编制与战舰等级 (军事/战斗系统)
人类以工业化优势快速量产舰队，但真正决定胜负的是指挥链、补给线与电子战能力。战舰等级越高越难维持，需要更高阶的航道权限与更残酷的资源抽取。

**Progression System:**

**Level 1: 护航舰群（矛）**
- Description: 以驱逐舰、护卫舰为核心，擅长护航、侦察与反走私，火力有限但数量可观。
- Requirements: 边境船厂可量产；需要基础反应堆与常规导弹库存。

**Level 2: 巡洋打击群（刃）**
- Description: 配备中型轨道炮与电子压制阵列，可进行航道伏击与中程舰队会战，是前线主力。
- Requirements: 获得潮汐晶核配额；指挥官通过“多节点会战”考核；至少一艘电子战旗舰。

**Level 3: 战列核心群（盾）**
- Description: 战列舰与航母组成的核心编队，能够进行行星封锁与大规模登陆支援，但对补给与维修极端依赖。
- Requirements: 控制一条深空干线航道；建立前进维修坞与反物质补给站；需要政治授权以动用战时征用令。

**Level 4: 泰坦级旗舰（冠）**
- Description: 单舰具备战区级指挥与灯塔干扰能力，可短时间“关闭”敌方窗口，改变整场战役节奏；同时成为各方暗杀与舆论战的中心。
- Requirements: 至少三座主灯塔供能；搭载潮汐回响核心；舰长需获得议会与军方双重任命。

**Level 5: 遗构融合舰（冕）**
- Description: 将遗构兼容层植入舰体，使其可进行无灯塔漂移与局部现实“重参数化”（例如扭曲传感器、削弱锁定）。代价是船员精神污染风险与不可逆的技术依赖。
- Requirements: 捕获完整遗构样本并完成三轮驯化；设立“梦境隔离舱”与心理防线；必须秘密进行以规避制裁协议审查。


### 人类联合体的权力结构：议会—财阀—军团 (政治/社会系统)
人类走出太阳系后形成松散联合体：议会负责合法性，财阀掌握工业与贷款，军团掌握武力与前线事实。三者互相挟持，使扩张既高效又充满背叛与交易。

**Progression System:**

**Level 1: 边境自治令**
- Description: 为提升殖民效率，前线获准自筹税收与治安，但也埋下军阀化与走私的种子。
- Requirements: 殖民地人口达到法定阈值；母星批准自治章程。

**Level 2: 战时征用体系**
- Description: 议会以战争为名征用企业产能与公民劳役，短期内暴涨军力，长期则激化阶层矛盾。
- Requirements: 遭遇外星封锁或航道危机；至少三次会战损失达到触发线。

**Level 3: 债务殖民模式**
- Description: 财阀以贷款与设备绑定殖民地，使其事实上成为企业领地；前线资源被抽走，叛乱成为必然。
- Requirements: 建立跨星系金融清算节点；控制关键船厂与补给站。

**Level 4: 军团直辖区**
- Description: 军方以“稳定”为名接管航道与殖民地，效率提升但人权崩塌；反对派转入地下，情报战白热化。
- Requirements: 发生重大叛乱或灯塔被夺；军团获得紧急授权并完成驻军部署。

**Level 5: 星冕协议（新秩序雏形）**
- Description: 主角团推动建立跨文明的新协议：航道主权、税制透明、战争限制与遗构监管。它可能成为人类的‘银河宪章’，也可能引发联合体内部的二次分裂。
- Requirements: 取得至少一个外星联盟签署；控制两条干线与核心灯塔群；公开揭露财阀与议会的关键黑幕以换取民意与军心。


### 遗构兼容层与“核心信号”污染 (悬疑/超科技设定)
遗构并非单纯科技，而像一套会自我保护的“文明接口”。它能提升航行与战斗能力，却通过梦境、幻听与集体共识偏移影响人类，仿佛在引导人类走向某条既定结局。

**Progression System:**

**Level 1: 残片接触**
- Description: 获得遗构碎片，能短暂提升传感器解析与引擎稳定，但会出现轻微共同梦境现象。
- Requirements: 从废墟星环打捞并通过生物隔离检疫；至少一名工程师完成接口翻译。

**Level 2: 兼容层试运行**
- Description: 舰船可调用“外来算法”进行航道修正与电子欺骗，代价是偶发‘记忆错位’与时间感偏差。
- Requirements: 建立兼容层沙箱；船员签署心理监测协议并接受梦境日志审查。

**Level 3: 共鸣驯化**
- Description: 兼容层与人类控制系统深度耦合，战时可实现局部优势（锁定干扰、窗口延长），同时出现“核心信号”片段，开始影响决策。
- Requirements: 完成三次战场验证；找到第一段核心信号密钥；建立反污染小组。

**Level 4: 门槛穿越**
- Description: 可进行无灯塔漂移并触碰‘折叠门槛’，但污染从个体扩散到群体：同一支舰队会形成近似宗教般的共同意志。
- Requirements: 夺取第二段密钥；进行一次牺牲性试航并存活；对外严格保密以避免政治清算。

**Level 5: 看门机制唤醒**
- Description: 当三段密钥齐备，遗构将向银河广播“完成讯号”，引来某种自动化的秩序修正者。它可能是古文明的防火墙，也可能是银河循环毁灭的执行者。
- Requirements: 集齐三段核心信号密钥；在核心潮汐区完成折叠；主角团必须在‘封存遗构’与‘借力称霸’之间作出不可逆选择。
