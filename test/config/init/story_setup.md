# 《风暴纪元：停车场的余烬》

## Story Setup

### Genre(s)
- 科幻灾难
- 悬疑惊悚
- 末世生存


### Core Premise
在近未来气候控制被滥用的都市中，'中央停车场'被设定为一种特殊的'锚点'，用于收容未被清除的'天候异常体'。一次被掩盖的'龙卷风'实验事故导致了该停车场的彻底物理毁灭，但风暴并未带来死亡，而是将无数人的'记忆与意识'剥离并注入到幸存者体内。主角作为唯一目击者，必须在城市崩溃前解开风暴背后的真相，同时面对自己记忆被篡改的危机。

### Theme
人类对自然的掌控边界、记忆与存在的意义、灾难中的共生关系。

### Story Rules/Constraints
- 所有'龙卷风'并非自然现象，而是'天候协议'的武器化应用。
- 被风暴摧毁的区域会进入'遗忘状态'，仅记忆与意识得以保留。
- 主角必须通过收集'记忆核心'来恢复自身人格，并重启天候锚点。
- 记忆提取需要承受'精神侵蚀'，提取越高，人格越不稳定。
- 停车场是唯一的'安全区'，一旦完全毁灭，全球气候将永久失衡。


### Target Audience
18-35 岁的科幻悬疑爱好者，喜欢硬科幻与人性探讨的读者。

### Tone/Style
冷峻、压抑、快节奏但有深度的叙事。

### Narrative Tense
过去式为主，穿插现在时增强紧迫感。

### POV Style
第三人称限制视角（聚焦主角内心与外部行动）。

## Storylines

### 风暴的余波
- **Type**: 主线剧情
- **Importance**: 1/10
- **Description**: 主要故事线，主角在停车场毁灭后的废墟中寻找幸存者与真相，逐步揭开天候控制的阴谋。

### 记忆幽灵
- **Type**: 辅助剧情
- **Importance**: 0/10
- **Description**: 次要故事线，主角体内不断浮现被抹去停车场的记忆片段，揭示过去发生的灾难级事件。

### 底层协议
- **Type**: 主线剧情
- **Importance**: 1/10
- **Description**: 隐藏剧情线，关于'天候协议'背后的真正目的是利用人类记忆作为能源，推动科技文明发展。


## Premises

### 天候管制系统 (世界观设定)
描述人类如何通过机械与科技控制天气，以及风暴与记忆之间的关联机制。

**Progression System:**

**Level 0: 观测者**
- Description: 能够监测风暴频率，识别异常，但无法干预。
- Requirements: 基础气象知识，观察能力。

**Level 1: 诱导者**
- Description: 能向特定区域发送'记忆脉冲'，制造小型局部风暴。
- Requirements: 通过'记忆核心'训练，掌握基础技术。

**Level 2: 压制者**
- Description: 可以稳定并压制大型风暴，防止记忆扩散。
- Requirements: 高强度精神训练，通过至少一次'记忆提取'。

**Level 3: 重构者**
- Description: 能修改风暴的'记忆频率'，将灾难转化为可利用资源。
- Requirements: 融合'记忆核心'与自身生物节，通过'精神侵蚀'测试。

**Level 4: 天候主宰**
- Description: 完全掌控天候系统，能创造或毁灭城市级的气候环境。
- Requirements: 牺牲部分记忆换取绝对控制权，成为'锚点'核心。


### 记忆实体 (力量体系)
描述被风暴摧毁后，残留的意识如何以'幽灵'形式存在及被利用的过程。

**Progression System:**

**Level 0: 碎片**
- Description: 零散的、痛苦的记忆片段，无法被提取利用。
- Requirements: 无。

**Level 1: 核心**
- Description: 经过风暴筛选的记忆核心，可被提取并注入他人。
- Requirements: 参与一次大型'记忆提取'事件。

**Level 2: 共生**
- Description: 记忆与宿主融合，宿主获得部分'记忆者'能力。
- Requirements: 通过精神融合实验，承受至少 50% 的人格侵蚀。

**Level 3: 容器**
- Description: 作为记忆存储库，能容纳大量'记忆幽灵'的集合体。
- Requirements: 成为'停车场锚点'的临时守护者。

**Level 4: 意识体**
- Description: 完全脱离物质形态，成为新的'风暴'，拥有自我意识。
- Requirements: 吞噬所有'记忆核心'，达到精神阈值极限。
