# 改装与风险

林渊 在 改装区 获得了 能量电池

# 第七章 改装与风险

改装间的空气里弥漫着烧焦的机油味和臭氧的刺鼻气息，每一次呼吸都像在吞咽带火的砂砾。林渊躺在冰冷的改装床上，全身连接着无数数据线，像是一只被无形巨兽缓慢吞噬的猎物。

“林渊，基因序列正在重组。”张杰的声音透过耳语器传来，冷静得不像话。他手里握着一块闪烁着幽蓝微光的以太电池，那是从废弃母舰上拆解的高能核心，也是这次改装的关键。

林渊的瞳孔微微收缩。他感觉到某种东西在血管里苏醒——那是共鸣基因深处涌动的力量。不仅仅是肌肉的酸痛，更是一种精神层面的震颤。视网膜上的系统面板开始跳动，一行行冰冷的数据刺入他的意识：驾驶员基因进化，Level 0 -> 1（感知级）。

“感觉怎么样？”张杰的手指悬停在插口上方，指节因用力而发白。

“很痛，但不是痛。”林渊低语，声音沙哑，带着金属摩擦的质感，“是……被唤醒。”

张杰的手指微微一抖，似乎有些犹豫。他看着林渊，眼神里闪过一丝复杂的挣扎。这种进化伴随着巨大的代价，一旦失控，肉体将瞬间崩解成废铁。但林渊的眼神里没有恐惧，只有猎人般的专注，死死盯着控制台那跳动的红色核心。

“启动。”

张杰咬了咬牙，按下确认键。以太电池瞬间连接上林渊的核心插槽，没有火花，只有电流无声地注入，如同血液汇入心脏。

嗡——

整个改装间震颤了一下。林渊感觉背后的虚空风暴规则被触发了，机甲核心等级系统 Level 0 -> 1（战术级）。不仅仅是升级，他感觉到周围的空气开始扭曲，仿佛有看不见的利刃在切割。

“这就是……新的力量吗？”张杰后退半步，警惕地观察着林渊，“你的反应速度，比之前快了不止一倍。”

林渊没有回答。他站起身，身上的金属外壳微微发热，发出低沉的轰鸣。他看向窗外的星空，那里不再是单纯的背景，而是充满了未知的危险。虚空风暴规则已经生效，他必须尽快找到它的源头。

“走吧，张杰。”林渊的声音里透出一股不容置疑的坚定，“我们要去狩猎。”

“等等！”张杰喊道，手里紧握着以太电池，指关节再次发白。

“没有以太电池，我动不了。”林渊冷冷地说道，他指了指自己的胸腔，那里有一颗跳动的心脏，比任何时候都更加沉重，仿佛要跳出胸膛。

林渊看着张杰，眼神里闪过一丝危险的光芒。他知道，张杰不是那种能轻易被说服的人，但这次，他需要对方陪在自己身边，直到风暴结束。

张杰深吸一口气，将以太电池紧紧贴在自己的胸口。他看向林渊，眼神复杂，最终化作一诺。

“我们……一起。”
