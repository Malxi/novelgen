# Story Outline

## 第一部：设定杀场的开卷

**Summary:** 网文作者林砚穿越成灵矿矿奴，发现矿场聚集了数十位同样穿越的作者，每人携带自己作品的金手指原型，却被“不可合作”铁律强制互杀。林砚以设定拆解与信息差为刃，从矿票与残页起步，逐步掌握结界的观测与判定逻辑，在一场场设定处刑中逼近离场资格，并触及“谁在写这本书”的黑暗真相。

### 第一卷：灵矿笔战

**Summary:** 从矿奴到筑基，林砚在无法结盟的规则下，利用交易记录、Flag惩罚、越界债务与因果钉书，连续击败多名作者对手，同时撬动狱卒体系的矛盾，最终抵达矿场核心入口，发现杀场真正的主人并非单纯修士。

#### 矿奴醒来

**Summary:** 林砚 在 灵矿场矿奴棚 发现了 自己成了最低层矿奴的事实

**Characters:** 林砚, 监工岑铁, 老矿奴周瘸子

**Location:** 灵矿场·矿奴棚与外矿道

**Events:**
- **goal** (林砚) [在杀场活过前三十日并摸清规则]: new
- **storyline** [主线：灵矿杀场的唯一结局]: started
- **storyline** [悬疑线：谁在写这本书？]: started

**Beats:**
1. 林砚在矿奴棚醒来，身体被矿毒侵蚀，记忆与现实错位
2. 监工岑铁宣读矿场规矩：外逃无效、冲突记录、矿票结算
3. 林砚听到“作者”暗语与陌生术语，意识到不止他一人穿越
4. 结界在他心底弹出“题眼触发”般的微弱提示，他强忍不去“宣告”任何目标
5. 他从周瘸子口中得知：有人靠金手指硬冲，死后当场“回收”

**Conflict:** 林砚既要隐藏自己是作者的身份，又要在资源极缺与随时冲突的矿场里活下去。

**Pacing:** normal

**Location:** 外矿道·第一采区

2. “题眼触发”显现：一次性“设定扫描”让他看到结界记录的冲突痕迹与矿票流向轮廓

**Location:** 外矿道·岔口与监控符阵下

**Events:**

- **relationship** (苏绾, 许禾) [临时结盟]: 尝试结盟后被结界判定合谋而崩解

**Beats:**

2. 两人共同对同一目标形成增益，结界判定“合谋”，反噬为矿毒暴涨与读心噪音

**Characters:** 林砚, 狱卒账房崔九, 周瘸子

2. 林砚用矿票换净毒药浴，避免炼体阶段暗伤，同时故意让交易记录看起来“无野心”

**Summary:** 林砚 在 矿车坡道 诱导了 韩策 自己触发系统反噬

**Characters:** 林砚, 韩策, 监工岑铁

**Location:** 外矿道·矿车坡道

**Events:**
- **item** (林砚) [矿票（来自韩策）]: get

- **goal** (林砚) [拿到第一份可验证的金手指代价样本]: achieved

**Beats:**
1. 韩策系统发布“限时任务：公开宣告击杀目标并完成”，逼迫他口头立Flag
2. 林砚故意用话术让韩策重复叙述计划，触发观测机制加码：任务条件被结界“严格化”
3. 林砚在矿车坡道制造可解释的意外：让韩策只能用蛮力强杀来抢矿票
4. 结界削弱数值碾压并引入第三方变量：岑铁临检，韩策为了“任务完成”越界出手
5. 系统判定失败与越界债务叠加，反噬使韩策当场吐血失明，林砚取走矿票并“合理收尾”

**Conflict:** 林砚必须在不直接蛮力杀人的情况下，让对手的设定自毁且结界认可结算。

**Pacing:** fast
