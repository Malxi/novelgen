# 《设定杀场：灵矿笔战》

## Story Setup

### Genre(s)
- 修仙
- 无限流
- 群像
- 智斗
- 异世界穿越
- 生存竞技
- 悬疑


### Core Premise
网文作者林砚一睁眼，穿越到修仙世界的灵矿场，成了最底层矿奴。他很快发现：不止他一个作者穿越，矿场里聚集了数十位来自不同题材的写手——玄幻、克苏鲁、系统流、种田、无限、诡秘、女频宫斗……每个人都获得了“自己小说里的金手指”。然而矿场上空悬着一条铁律：作者之间无法结盟、无法共享资源、甚至无法在同一目标上形成协作效果；他们必须彼此战斗，直到只剩最后一人才能离开矿场。更残酷的是，金手指强到离谱，硬碰硬只会同归于尽，胜负只能靠智斗：用设定漏洞反制设定、用叙事逻辑扭转局势、用信息差逼迫对手自毁。林砚要做的不是写出爽点，而是在“设定杀场”里活成唯一的结局。

### Theme
创作者与规则：当“设定”成为武器与牢笼，真正决定生死的不是力量，而是理解规则、承担选择与保持人性的能力。

### Story Rules/Constraints
- 矿场为封闭结界“杀场”，外逃无效；死亡者的金手指与部分“设定权重”会回流至杀场核心。
- 所有穿越者均为网文作者，并自动绑定其代表作的“金手指原型”。金手指会随“叙事权重”成长。
- 硬性禁令：作者之间无法合作。具体表现为：组队契约无效、共享增益失效、治疗/护盾/分摊伤害对“他人”无效，联合施法会被结界判定为“合谋”并反噬。
- 战斗必须以“智斗”结算：当双方进入冲突，结界会强化“信息、逻辑、因果”对胜负的影响，削弱纯数值碾压；若一方仅靠蛮力强杀，将触发等价反噬或强制引入第三方变量。
- 每个金手指都有“设定边界”。越界使用会产生“自洽债务”，债务累积到阈值将被结界强制修正（例如：能力失控、反向改写、人格崩坏、剧情回收）。
- 矿场资源有限：灵矿、矿票、功法残页、狱卒情报皆可作为筹码，但交易只允许“一对一”完成，且交易内容会被结界记录，成为可被推演的公开变量。
- 结界存在“观测机制”：越像写作（立Flag、立誓言、宣告目标、重复叙述），越容易被结界捕捉并加码惩罚或利用；沉默与伪叙事是生存技巧。
- 最终胜利条件表面为“活到最后”，但隐藏胜利条件与矿场主人的真实目的有关。


### Target Audience
18-35岁偏男性读者为主，喜欢修仙升级、群像对决、设定流智斗、无限生存与悬疑反转的网文读者。

### Tone/Style
高压冷峻、智性对抗、设定流爽感与黑色幽默并存；胜利带代价，反转频密但遵循逻辑自洽。

### Narrative Tense
第三人称过去时

### POV Style
多视角限知（以林砚为主，关键章节切换到对手视角呈现反制逻辑与信息差）

## Storylines

### 主线：灵矿杀场的唯一结局
- **Type**: 主线
- **Importance**: 10/10
- **Description**: 林砚从矿奴起步，在无法结盟的前提下与各路“作者金手指”对抗。他通过拆解对手设定边界、制造叙事陷阱、诱导对手越界背债，逐步积累叙事权重，逼近“离场资格”。随着对手减少，结界规则加压，最终指向矿场主人与杀场存在的真实原因。

### 副线：禁令的漏洞与“不可合作”的真义
- **Type**: 规则推理/智斗
- **Importance**: 9/10
- **Description**: 林砚发现“不能合作”并非绝对无法影响他人，而是无法形成“共同收益”。他研究结界判定逻辑：如何通过误导、交换、第三方触发、对赌条款，让对手在“自愿”与“自洽”中走向必败。此副线推动智斗升级与规则反转。

### 副线：矿场势力与狱卒系统
- **Type**: 权谋/社会生态
- **Importance**: 8/10
- **Description**: 矿场并非纯竞技场，狱卒、监工、矿主的修仙体系真实存在。他们利用作者互杀产出的“设定残渣”炼器炼魂。林砚必须在不结盟的情况下，操控狱卒矛盾、借刀与反借刀，争夺矿票与情报，撬动更高层资源。

### 角色弧光：创作者的底线与写作之罪
- **Type**: 人物成长
- **Importance**: 8/10
- **Description**: 每位作者都曾在作品里“安排”过无数角色的苦难，如今他们亲自承受。林砚从“只求活命”走向“拒绝成为矿场主人需要的怪物”：他在胜利策略中不断选择更难但不自毁的路径，探讨创作者责任与人性底线。

### 悬疑线：谁在写这本书？
- **Type**: 悬疑/元叙事
- **Importance**: 9/10
- **Description**: 杀场会对“叙事”做出响应，像有一个看不见的编辑在改稿：伏笔被回收、巧合被矫正、角色被强行对称。林砚追查“观测机制”的源头，发现矿场主人可能不是修士，而是某种以故事为食的存在。

### 对手群像线：金手指的代价清单
- **Type**: 群像/对决
- **Importance**: 9/10
- **Description**: 每个作者的金手指都强，但都有致命副作用：系统任务反噬、无限复活的记忆崩坏、克苏鲁认知污染、宫斗读心导致人格分裂等。林砚不靠碾压，而是通过识别代价、制造触发条件，完成一场场“设定处刑”。


## Premises

### 修仙境界：矿奴修行路 (力量体系)
在灵矿场环境下修行，灵气杂驳且含矿毒，进境快但代价大。境界不仅决定战力，也决定能否承受更高级“设定权柄”的负荷。

**Progression System:**

**Level 1: 炼体·矿奴境**
- Description: 以矿毒淬体，筋骨强于凡人，能短时间爆发蛮力与耐受。此阶段更像“活下去”的门槛。
- Requirements: 连续采矿三十日不倒；以“矿票”换取一次净毒药浴，否则易留下暗伤。

**Level 2: 炼气·杂灵境**
- Description: 能引杂灵入体施展基础法术，但法力不纯，容易走火或被他人以“属性克制”针对。
- Requirements: 获得一本残缺吐纳法；在矿洞深处完成一次“闭气采脉”以建立灵息循环。

**Level 3: 筑基·毒火台**
- Description: 以矿毒与地火凝成“毒火台基”，法力质变，能承受更高强度的设定冲击；同时也更容易被结界记录为高威胁目标。
- Requirements: 集齐三种矿髓（寒髓/赤髓/青髓）以平衡毒火；筑基时需在狱卒眼皮底下完成，必须设计遮掩。

**Level 4: 金丹·裂稿丹**
- Description: 金丹如“压缩的叙事核心”，可将一次战斗中的关键因果压成“裂稿”爆发，短时间改写局部战局。
- Requirements: 完成一次高难智斗胜利并不触发越界反噬；夺得“设定残页”用于凝丹，自洽债务不得超过阈值。

**Level 5: 元婴·执笔婴**
- Description: 元婴如分身亦如“作者化身”，可在短时间进行多线程推演与布置，成为智斗的极致形态；但一旦被观测机制锁定，将遭遇强制剧情修正。
- Requirements: 找到观测机制的一个可利用盲点；以自身最痛的秘密作为“代价素材”完成元婴凝形（代价不可回避）。


### 作者金手指：叙事权柄（设定之笔） (核心机制/智斗系统)
每位作者的金手指源自其代表作，但本质被统一为“叙事权柄”：你越能让行为符合逻辑自洽、信息闭环、代价等价，权柄越强。反之，越靠嘴炮与无代价外挂，越快背上自洽债务。

**Progression System:**

**Level 1: 引子：题眼触发**
- Description: 只能触发作品金手指的一个“标志性效果”（例如：一次性扫描、一次保命、一个小系统提示）。效果强但次数少，且会留下可被追踪的叙事痕迹。
- Requirements: 在首次生死危机中做出“符合原作主角性格”的选择以完成绑定。

**Level 2: 章法：设定具现**
- Description: 可将原作中的一个明确设定具现到现实（如道具、规则片段、触发条件），但必须同时具现其代价或限制。
- Requirements: 用一次智斗胜利证明“你理解自己的设定边界”；向结界提交一次等价代价（失去某项资源/记忆片段/寿元）。

**Level 3: 反转：因果钉书**
- Description: 能把对手的一个行为钉成“既定因果”，迫使其在后续兑现（例如：你诱导他立誓、立Flag、签交易条款），成为高阶智斗核心能力。
- Requirements: 让对手在自愿状态下说出或完成可被记录的承诺；同时自身不得出现更大的逻辑漏洞，否则反噬回钉自己。

**Level 4: 高潮：叙事换轨**
- Description: 在不违背大规则的前提下，将一段局势从A逻辑换到B逻辑（例如把“战斗”换成“审判”、把“追杀”换成“交易清算”），从而改变胜负判定方式。
- Requirements: 制造三重信息差（对手以为的规则/你掌握的规则/结界实际判定规则）；并支付一次“自洽债务利息”（永久损失一项金手指小功能）。

**Level 5: 终章：署名权（临时主角）**
- Description: 短时间夺取局部叙事中心，成为“临时主角”，让概率、巧合、路人变量更倾向你；但结束后必有等价回收：你将失去某段人生的真实感或情感能力。
- Requirements: 连续三场对决不靠蛮力取胜且不触发越界；找到并利用观测机制的核心接口一次（成功一次，失败即被剧情回收）。


### 灵矿资源体系：矿票—矿髓—设定残页 (资源/经济与风险系统)
矿场的硬通货不只是灵石，而是能让金手指升级的“设定残页”。灵矿越高级，越可能产出残页；但越深入矿脉，越接近杀场核心与观测机制。

**Progression System:**

**Level 1: 矿票**
- Description: 矿场最基础的结算凭证，可换食物、药浴、工具与低阶功法抄本。矿票流向可推演，暴露你的位置与状态。
- Requirements: 完成日采额度；或通过单挑夺取他人矿票（夺取会留下冲突记录）。

**Level 2: 灵矿砂**
- Description: 含杂灵之砂，可用于炼气、布阵、设置一次性陷阱。对智斗者而言是“布局材料”。
- Requirements: 在矿洞支线脉完成一次“净砂筛选”，避免矿毒超标。

**Level 3: 矿髓**
- Description: 灵矿精髓，能稳定筑基、修复自洽债务带来的身体损伤，也是狱卒层的管控资源。
- Requirements: 进入深层矿脉；必须通过狱卒审查或用信息差绕开巡查。

**Level 4: 设定残页**
- Description: 杀场真正的“成长材料”，可喂养叙事权柄、修补金手指边界、或用来交易换取关键情报。残页越多，你越接近被观测机制锁定。
- Requirements: 击败并“合理收尾”一名作者（结界认可的结算）；或从矿场核心的废稿堆中偷取（风险极高）。

**Level 5: 废稿核心（未署名章节）**
- Description: 传说中的顶级资源，能短暂让你改写一条矿场子规则（例如：取消一次合作禁令判定、豁免一次越界反噬）。但改写越大，回收越狠。
- Requirements: 找到杀场核心入口并通过观测机制的“审稿”；提交一个能让结界满意的“代价结局”（通常意味着牺牲自我重要之物）。
